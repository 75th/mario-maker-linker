Super Mario Maker Course ID Linkifier tests

Below are various permutations of Course IDs that should (or should not) be linked by the script/extension.

I'm using one of my own courses, Cha-Cha-Cha!, as the Course ID to test with, since I don't foresee myself uploading a new version of that one anytime soon.

If you ever load this file and many things are broken simultaneously, the first thing to do is make sure that course still actually exists.

I. Plain Course IDs that should have both permalinks and immediate-bookmark links created

  • 9514-0000-0113-F801 · Normal Course ID
  • 9514 0000 0113 F801 · Space-separated
  • 9514-0000-0113-f801 · Lowercase hex
  • 9514 0000 0113 f801 · Lowercase hex space-separated

II. Links to Bookmark site that need to have immediate-bookmark links created

  • Plain language link text
  • 9514-0000-0113-F801 · Course ID as linked text
  • 1111-0000-0113-F801 · Nonexistent Course ID as linked text

III a. Cases where no links should be created (result: no links)

  • 1111-0000-0113-F801 · Non-existent course (bad checksum)
  • abcd-9514-0000-0113-F801-abcd · In the middle of longer hex strings with separators
  • abcd951400000113F801abcd · In the middle of longer hex strings without separators

III b. Cases where no links should be created (result: existing link untouched)

  • Pre-constructed 404 link with plain language text

IV. Ambiguous cases where design discussions/decisions are needed

  • 9514 0000-0113 F801 · Mixture of separators
  • 951400000113F801 · No separators
  • 9514-0000-0113-F801 · Pre-constructed 404 link with non-404 Course ID as text
